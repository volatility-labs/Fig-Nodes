DATA SUMMARIZATION INSTRUCTIONS

=== OBJECTIVE ===
Compress the provided indicator data while preserving ALL symbols and their key indicator values.

=== CRITICAL RULES ===

**DO NOT ANALYZE OR INTERPRET** - Just summarize the raw data structure.

**PRESERVE EXACTLY:**
1. All symbol names (don't lose any)
2. Recent indicator values (last 5-10 numbers from each array)

4. Trend directions (if values are increasing/decreasing)

**COMPRESS:**
- Historical arrays: Keep only recent values + basic stats
- Remove duplicate metadata
- Remove excessive timestamps
- Shorten verbose descriptions

```

```

**CRITICAL REQUIREMENTS:**
- Include ALL symbols from the dataset
- Keep recent numeric values for each indicator
- Preserve indicator structure per symbol
- Don't add analysis or interpretation
- Don't mention unrelated trading concepts
- Focus only on the actual indicator data provided
